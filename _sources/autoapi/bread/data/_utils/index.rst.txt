.. py:function:: dump_npz(filepath, data: dict, force=False, dry=False, compress=True)

   Dumps data to disk in .npz format

   :param filepath: filepath of the .npz file
   :type filepath: str
   :param data: data to dump. each item of the dict should be an array-like
   :type data: dict
   :param force: allow overriding files. Defaults to False
   :type force: bool, optional
   :param dry: whether to actually write data. Defaults to False
   :type dry: bool, optional
   :param compress: whether to compress the data. Defaults to True
   :type compress: bool, optional
